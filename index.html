vue-seamless-scroll

A simple, Seamless scrolling for Vue.js.

Get Started →

Initial configuration support

current support for seamless scrolling, single-step scrolling, and manual switching support for horizontal direction.

Compatible with multiple platforms

IE9+、Firefox、Chrome、Safari、iOS、Android。

Multi-technology stack version support

Currently supports Vue2 and JavaScript versions.
